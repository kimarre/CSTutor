CSTutor Integration Test Plan

After each individual tool has been tests, the integration process will be as follows:

  1. User functionality will be combined with Chat; run tests to verify users within the chat program.
  2. Database functionality will be combined with User and High Level Manager; data can be loaded from and saved to the database.
  3. Progress funtionality uses User status as verification and identity; run and verify Progress tests.
  4. Tutorial, Progress, and Quiz functionality will be integrated with the backend database; Run all tests verifying the three sub-tool functionalities.
  5. Tutorials, Progress, and Quizzes will test that any shared components are compatible with one another; rerun all three individual sub-tool tests.
  6. The High Level Manager will integrate links to Tutorials and Quizzes; rerun all sub-tool tests.
