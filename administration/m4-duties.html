Simon Vurens Deleting Objects Select an object in the manager tree, then click 'Delete' Text will appear in the console with what type of object is getting deleted.
Simon Vurens Adding Objects Click on 'Add...' and add an object. You must select an object at most one level higher to add a new section/unit/tutorial. (To add a unit, you must select a unit, tutorial, or section before clicking 'Add...') The object will be created and placed in the tree.
Simon Vurens Saving Objects Select the 'save' button The data will be saved.
Simon Vurens Adding students to a class Select 'New...' from the manager screen. Ensure that 'Class' is selected from the dropdown. Click 'Add' then click a student, then click 'OK' Text will appear in the console with the added student's name. They are not yet added to the model.
Kyle Reis User Registration Select Login from the menu bar. Click Register on the Login screen. On the Registration screen fill in the information (the email field does get validated). Press submit upon filling the form out. When submit is pressed a system out will print either that the registration occured or that it failed.
Kyle Reis Login Select Login from the menu bar. Fill in the login info with the info that you registered previously (only works if still in the same session) and click Login. If the information is incorrect then an error will appear in the dialog and the inputs will clear. If the information is correct then a syste out will print the users full name.
Kyle Reis Password Reset Select Login from the menu bar. Click "Forgot Password?" on the login screen. Type in a registered email address (only works if still in the same session as when registered) and click ok. If the email address is incorrect an error will appear on the screen. If the address is correct then a dialog will pop up saying a token has been emailed along with the system out printing the token (not encoded as planned due to some Java issues) and that the token has been emailed.
