Chat package contains the models for the online chat and online status windows. ChatOverlay contains the model for the buttons at the top level GUI. ChatWindow contains the models for the next box and console. Console contains a model that represents a collection of strings that will be output to the screen. OnlineWindow contains the model that shows which students are online and which aren't. Student is a model that represents a student. Student text helps display information about the student in the console. And TextBox is the text box where the user inputs data for the console to display.
