1.5.1. W3Schools

W3Schools is an online reference tool for most web development based tools including HTML/CSS, JavaScript/JQuery, SQL, and PHP. It's a great tool for both beginner and advanced programmers alike because the site offers information in a logical order to learn. An organized sidebar of topics allows more advanced users to jump straight to their topics of interest, while beginners may start from the top and work their way down to the more advanced topics.

W3Schools is perhaps known most for their direct code examples. Throughout the site, code examples reside in text boxes with "Try it yourself" buttons that let you run the code and compare the code to what the end result turns out as.

Good Features:

  • Provides working code examples and allows you to see the output
  • Web forums that allow users to communicate with one another about problems and conventions
  • The well-organized subject/topic sidebar allows you to explore the topics of interest easily

Bad Features:

  • Directs you to another page that runs your code
  • Only supports web development based languages
  • Some information given is inaccurate
  • Users can’t contribute new material

Missing Features:

  • No interactive exercises
  • No support for any languages that aren’t web based
  • No community based support and/or interaction
  • No way to track your own status or progression in learning
