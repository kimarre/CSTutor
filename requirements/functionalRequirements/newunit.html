2.2.5. Adding a new unit to a tutorial and moving tutorials into units

From the manager screen, the user clicks on the 'New Unit' button. The manager shows the dialogue to create the unit.


The user simply enters a name for a unit.


The user presses 'OK.' The manager then takes the user over to the quiz or tutorial editor, depending on what page type was selected. After the user returns from the editor, the manager shows the page in the list.


To move an existing tutorial into the newly created unit, the user selects the tutorial they want to move. In this case, the user selects the 'Hello World' tutorial by clicking on it in the list. The user then clicks on the 'Move...' button. The manager shows the move dialogue to the user.


The user then selects which unit to move the tutorial to, using the dropdown menu with a list of all available units.


The user then presses the 'OK' button. The tutorial will be moved to inside the unit.
