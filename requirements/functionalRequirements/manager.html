2.2. Manager View

Manager view allows the professor to manage all of his or her classes quickly and efficiently. The scenarios in this section show a professor navigating the manager screen. Scenarios include:

  • Creating a new class
  • Creating a new section
  • Creating a new tutorial
  • Creating a new page in a tutorial
  • Creating a unit and moving an existing tutorial into it
  • Reordering elements in the manager


There are also simple navigational buttons that allow navigation to other parts of the cstutor. For example, the 'Hold Office Hours' button takes the user to the online office hours page.
