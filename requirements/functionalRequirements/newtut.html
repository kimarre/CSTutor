2.2.3. Adding a new tutorial

The user is on the manager screen with a class already created. In this case, there is also a section. The user clicks on the 'New Tutorial' button. The manager pops up the dialogue to create a new tutorial.


The user then fills in the name of the tutorial, and selects if it should be part of a larger unit from the dropdown menu. In this case, there are no units to choose from.


The user then clicks the 'OK' button, and a new tutorial is created.
